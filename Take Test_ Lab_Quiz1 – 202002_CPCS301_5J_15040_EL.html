    Rewrite the following C code in PHP, taking into consideration preserving the same scope for the variable “number”. The code read a number and test whether it is divisible by 3 or not. Your PHP code must include “DivisibleBy3” function that does not take any parameters and return an indicator that show whether the number is divisible by 3 or not as in the C code. Note: add your name, id and section as a comment in the code and name your file “Q2_ID_Section.txt or .doc”

    #include <stdio.h>

    int number;

    int DivisibleBy3()

    {

        if(number%3==0)

            return 1;

        else

            return 0;

    }

    void main()

    {

        printf("Enter an integer number: ");

        scanf("%d",&number);

        if(DivisibleBy3())

        printf("%d is divisible by 3 ",number);

        else

        printf("%d is not divisible by 3 ",number);

    }


    Sample Output1:
